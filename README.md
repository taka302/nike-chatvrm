# Nike ChatVRM Enhanced 🎭

キャラクターと会話しながら、自動で関連する画像・動画を表示してくれるAIチャットアプリです。

## ✨ 機能

- 🎭 **キャラクター変更**: 3種類のキャラクターから選択可能
- 🗣️ **音声入力**: マイクで話しかけられます
- 🔊 **VOICEVOX対応**: AIの返答を音声で読み上げ
- 📸 **自動画像表示**: 会話に合わせて関連画像を自動表示
- 🎥 **自動動画表示**: Pexelsから無料動画を取得
- 💬 **ChatGPT連携**: 自然な会話が可能

## 🚀 使い方

### 1. ファイルの準備
すべてのファイル（index.html, style.css, script.js, config.js, README.md）を同じフォルダに入れます。

### 2. APIキーの取得

#### ChatGPT APIキー
- 既にお持ちとのことなので、そのまま使用してください

#### Pexels APIキー（無料）
1. https://www.pexels.com/ にアクセス
2. 無料アカウント作成
3. https://www.pexels.com/api/ でAPIキーを取得

### 3. アプリの起動
1. `index.html` をブラウザ（Chrome推奨）で開く
2. 右上の「⚙️ 設定」ボタンをクリック
3. ChatGPT APIキーとPexels APIキーを入力
4. 「保存」をクリック

### 4. 使ってみる
1. キャラクターを選択
2. メッセージを入力して「送信」
3. 自動で画像・動画が表示されます！

### 5. VOICEVOX連携（オプション）
1. VOICEVOXをダウンロード: https://voicevox.hiroshiba.jp/
2. VOICEVOXを起動しておく
3. アプリが自動で音声読み上げします

## 💡 使用例

**質問**: 「富士山について教えて」
→ AIが富士山について説明 + 富士山の画像・動画が自動表示

**質問**: 「猫の可愛い写真見せて」
→ 猫の画像が複数表示されます

## ⚠️ 注意事項

- インターネット接続が必要です
- ChatGPT APIは有料です（使用量に応じて課金）
- Pexels APIは無料です
- 音声入力はChromeブラウザ推奨

## 🆘 トラブルシューティング

### メッセージが送信できない
→ 設定でAPIキーが正しく入力されているか確認

### 画像が表示されない
→ Pexels APIキーが正しいか確認

### 音声入力できない
→ Chromeブラウザを使用し、マイクの許可を出してください

### VOICEVOXが動かない
→ VOICEVOXアプリが起動しているか確認

## 📝 カスタマイズ

### キャラクターを追加したい
`script.js` の `characters` オブジェクトに追加できます。

### デザインを変更したい
`style.css` を編集してください。
